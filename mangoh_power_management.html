MangOH Power Management

  May 2018 - Dec 2018
  Richmond, BC
As a side project at Sierra Wireless, I developed a Power Management system using C and Lua, that monitors current  consumption at each mode (Eg. Data transmission, connecting to server, ultra-low power mode) of the mangOH IoT device. The program then analyzes the data and predicts battery capacity to run the operation for a given period of time. The program  also extracts power statistics and provides suggestions on reducing power consumption. One foreseen use-case behind this  project is asset tracking for freight trucks.
