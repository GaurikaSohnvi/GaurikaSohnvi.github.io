Antenna Tracking Station

Sep 2018 - Apr 2019
 Vancouver, BC
Led a team of 8 to develop an antenna tracking station. The station was programmed using Python on a Raspberry Pi to continuously orient itself towards the aircraft. Its open-loop control uses GPS data from the station and aircraft to continuously calculate the new heading. With 2DOF, the station can be manipulated in yaw and pitch directions. This project reduced flight mission time by 30% as it allowed long range live image transmission to the base station. This was built as a project for the UBC Unmanned Aircraft Systems design team’s ground control station when I was the Radio Communications Technical lead.

The Helping Hand

Jan 2020
 Vancouver, BC
This hackathon project was created for situations that present danger to the human physically present. The Helping Hand uses hand gesture recognition to simulate a human arm through the robotic arm. Paticularly, I worked on the robot kinematics and control. To simulate hand gestures, OpenCV was used to track a green ball via camera. The project was further developed when gesture recognition was performed using Intel RealSense’s point cloud depth perception and Tensor Flow’s CNN library. This project won the Top 20 award out of 145 teams.

MangOH Power Management

  May 2018 - Dec 2018
  Richmond, BC
As a side project at Sierra Wireless, I developed a Power Management system using C and Lua, that monitors current  consumption at each mode (Eg. Data transmission, connecting to server, ultra-low power mode) of the mangOH IoT device. The program then analyzes the data and predicts battery capacity to run the operation for a given period of time. The program  also extracts power statistics and provides suggestions on reducing power consumption. One foreseen use-case behind this  project is asset tracking for freight trucks.
